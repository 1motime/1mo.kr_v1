Overview

Value Valley는 정보를 체험하며 이해할 수 있는 인스톨레이션 작품이다. 일반적인 정보디자인 작품은 인쇄물과 모니터 출력 기반 그래픽 작업이 대부분이다. Value Valley는 공간에서 조형물과 인터랙션한다. 정보를 물리적으로 느낄 수 있는 실험적인 프로젝트다. 관객들은 자신의 가치관이 반영된 산맥을 직접 만들어보는 새로운 경험을 할 수 있다.

Structure

'Information Wall'에서 가치관 형성 과정을 이해한다. 어린시절부터 현재까지 가치관 단계를 실루엣과 함께 설명한다. 직접 자신의 모습이 투영된 실루엣으로 몰입감을 높인다. 'Information Table'에서 가치관 유형을 알 수 있는 산맥을 만든다. 가치관 정량화 척도는 가치 심리학자 Schwartz의 Value Survay다. 이 설문은 가치관유형별 정량화된 데이터 수집을 할 수 있다.

(왼)가치관 형성 과정. 인간의 가치관은 일반적으로 만 5세부터 21세까지 총 3단계를 거쳐서 형성된다.
(오)Schwartz Value Servay 기준 현재 자신의 가치관 척도. 키워드 설문을 통해 정량화 할 수 있는 척도다. 각 키워드의 점수를 합산해서 가치관 유형을 알 수 있다.

User Journey Map

'Information Wall'에서 사용자 실루엣을 타임라인에 함께 보여준다. 자신의 모습을 투영함으로서 몰입해서 가치관 형성과정을 이해한다. 'Interactive Table'에서 현재 가치관 유형을 파악한다. 10개 설문에 답을 입력하면, 데이터는 산맥 모습으로 반영된다. 설문 결과는 엽서 형태로 출력된다.

Application
Interactive Table Making Process

설문 데이터를 산맥 형태로 맵핑할 수 있는 특수 제작 테이블이다. 데이터 수치에 따라 높이가 반영되어 움직이는 구조다. 직접 설계하고 제작했다. 고장나서 수리해야하는 상황을 대비해서 접착제를 일체 사용하지 않았다. 볼트와 자석을 이용한 조립식으로 제작했다.
